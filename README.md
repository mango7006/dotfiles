## Installtion guide for dotfiles
Clone my dotfiles
```bash
cd ~
mkdir -p gits
cd ~/gits
git clone https://github.com/mango7006/dotfiles
cd dotfiles
```

Install dependencies
```bash
./installPKGS.sh
```
